# SF Crime COVID Story

Short narrative visualization for Exercise 2.1.

## Build the assets

From the repository root:

```bash
python3 sf-crime-covid-story/scripts/build_story_assets.py
```

This exports exactly three visualizations:

- `sf-crime-covid-story/visualizations/static-chart/monthly-crime-trend.png`
- `sf-crime-covid-story/visualizations/map/district-pandemic-shift.html`
- `sf-crime-covid-story/visualizations/interactive/category-shift-heatmap.html`

The page itself is `sf-crime-covid-story/index.html`.
